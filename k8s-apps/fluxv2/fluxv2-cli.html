FluxV2 CLI

Install the flux cli

curl -s https://fluxcd.io/install.sh | sudo bash

Enable Autocompletion - Bash

vi ~/.bashrc

# Add this line
. <(flux completion bash)

Enable Autocompletion - Zsh

vi ~/.zshrc

# Add this line
source <(flux completion zsh) && compdef _flux flux
